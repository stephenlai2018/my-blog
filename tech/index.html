# 使用 VuePress 建立個人網站

長久以來一直想要建立個人網站, 寫部落格, 除了沒有很強烈的動機, 網站開發工具的不親民 / 不便利佔了很大因素. 所以遲遲沒有自己的專屬園地. 在專門為產生靜態頁面網站的開發工具出現前, 想要開發個人網站, 必須要學習 html 標籤, css 樣式, 如果要加入動態效果, 還要學習 javascript. 用寫程式的方式寫部落格, 我個人是覺得很不對勁, 因為注意力都是放在程式的語法規則, 內容反而變成其次. 讓原本只是寫日誌這麼單純的事門檻變得很高, 當然會讓很多人打退堂鼓. 所以雲端服務例如 medium, quip, iT邦幫忙, 幫趣等讓使用者可以輕易發文的服務非常受歡迎, 因為只需要在框框內填入文字, 上傳圖片 / 影片, 然後按下發佈鍵就行了.

# 到底是寫網誌還是開發 App ???

用寫網頁的方式寫網誌, 就像在開發一個 web app, 首先文字前後都要被標籤包住, 像這樣

<h1>這是標題</h1>

<h2>這是副標題</h2>

<p>這是內人</p>

lorem ipsum lorem ipsum lorem ipsum lorem ipsum lorem ipsum lorem ipsum lorem ipsum

對於工程師來說這樣看起來好像還好, 寫網頁不就是這樣嗎? 不過前後文被標籤包住滿奇怪的, 如果要更改標籤會變得很麻煩, 而且 </> 符號常常會被誤刪除或被忽略. 如果還要給樣式, 會變成這樣 :

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam voluptatem autem suscipit temporibus dolorem repellendus sunt delectus esse magni quidem obcaecati xcepturi repudiandae perferendis?
</p>
<style>
p {
  border: 1px solid green;
  color: red;
  font-style: italic;
  text-shadow: 2px 2px 2px #888;
  font-size: 1.2rem;
}
p:hover {
  color: magenta;
  font-size: 1.5rem;
}
</style>

以上就是開發網頁的方式, 因為 html 檔案本來就可以讓我們寫程式, 所以通常會寫 css 給標籤樣式, 讓網頁呈現豐富的顏色 / 影字 / 邊框等效果, 還可以引入第三方程式庫, 如果希望產生動態效果, 還要用 javascript 編程. 不知不覺中就會寫出很多代碼, 讓原本只是要寫純文字內容失焦 / 變調. 這樣是在寫程式還是寫日誌 ???

# Static Site Generator / 靜態網站產生器

也因為以上的原因, 一些專門為單純寫文字的工具應運而生, 例如知名的 Hexo, GitBook, Docsify / Docute, Nuxt 等工具應運而生. 以上工具的優劣點在 VuePress 官網都有詳細說明. VuePress 是專門為開發靜態網站 ( 以文字為主 ) 打造的開發工具. 讓建立個人網站變成一件輕鬆愉快, 容易上手的事情. VuePress 採用 markdown 語言 副檔名 .md, markdown 語言的特性是簡潔 / 易學, 就是方便開發者寫內文, 插入圖片, 清單, 網站連結等靜態內容. 沒有 html 的諸多規則, 不用寫程式.

# Markdown VS HTML

markdown                                      html
-----------                                   -------------------
# 這是標題                                     <h1>這是標題</h1>

- 蘋果                                         <li>蘋果</li>
- 橘子                                         <li>橘子</li>
- 芭樂                                         <li>芭樂</li>>

[進入 Google](https://www.google.com.tw)       <a href="https://www.google.com.tw" target="_blank">進入 Google</a>

![](https://www.photo.com/myPic)               <img src="https://www.photo.com/myPic">

markdown 語言和 html 相比在視覺上是不是簡潔洗練許多呢 ? 沒有標籤看起來是不是很清爽呢 ???

除了 markdown, VuePress 允許在 md 頁面使用 html 標籤, css 樣式和 javascript 編程, 有些人會覺得純文字太單調, 想要美化頁面. 而 VuePress 滿足這一點, 所以你也可以把寫網頁的方式應用在 md 檔案.

# 摘要

  • 建立專案 : npx create-nuxt-app

  • 啟動專案 : npm run dev

  • 程式進入點 : server/index.js

  • pages 資料夾存放頁面 ( Nuxt 會將這個資料夾內的檔案編譯成路由檔案, 我們不用寫 router )

  • layouts 資料夾存放導覽 / 側邊攔

  • components 資料夾存放組件
